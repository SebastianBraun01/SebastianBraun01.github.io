LED-Mesh-Netzwerk

Raumbeleuchtung basierend auf den Neopixel-LEDS (WS2812B), die man gemütlich von seinem Smartphone aus bedienen kann. LED-Netzwerk kommuniziert über das Bluetooth Mesh Netzwerk von Particle. Das Netzwerk besteht aus mehreren einzelnen LED-Streifen, einer musikreaktiven LED-Wand und einer einem zentralen Access Point.

Steckbrett-Computer

8-Bit Computer, konstruiert auf einer Reihe von Steckbrettern, der die grundlegende Funktionsweise von Computerhardware veranschaulichen soll. Der ganze Computer besteht nur aus simplen ICs der 74er Serie TTL-Chips und jeder Teil ist mit einer Reihe von LEDs ausgestatted, die die gespeicherten Daten anzeigen.

Nixie-Uhr

Dekorative Uhr aus alten russischen Nixie-Röhren (Kaltkathodenröhren). Die komplette Uhr besteht nur aus einer Platine, die das spezielle Netzteil und die ICs zur Ansteuerung der Röhren beinhaltet.
